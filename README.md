# webpack4.x + vue2

之前已经做过 [webpack4.x][1] 相关的配置详细步骤，此次会再结合进 `vue` 来进一步配置我们的项目


## 实现内容

* vue 路由实现
* vue 模块化
* mockjs 模拟后端数据
* axios 获取后端数据

## 命令

1. 安装

```
npm install
```

2. 运行开发环境

```
npm start
```

3. 输出生产环境

```
npm run build
```
